== Description ==
Related Products for WooCommerce allows you to choose Related products for the particular product.

&#9989; Tested OK with WooCommerce 3.8.0

= Plugin Functionality: =
* Add Related products.
* Display chosen Related products on product detail page instead of display same categories products.

== Installation ==
* Download the plugin
* Upload the folder "Related-product-for-woo" to wp-content/plugins (or upload a zip through the WordPress admin)

== Frequently Asked Questions ==
= How to Setup : =
* Step-1: Go to edit product section in product data section go to Linked Products you will find 'Related Product'.
* Step-2: Select products which you want to set Related products for that product.
* Step-3: Save product.
